## 文件功能说明

### 1. 入口模块

#### main.py
主程序入口，协调经典和量子散射模拟的完整流程：
- 设置物理参数和网格参数
- 验证Nyquist条件
- 均匀抽样30个粒子的碰撞参数（b: 10~125 fm）
- 调用经典散射和量子散射模块进行数值模拟
- 保存模拟数据并生成可视化图表

### 2. 物理/算法模块

#### 物理参数.py
集中定义所有物理常数和实验参数：
- 基本常数：电子质量、α粒子质量、约化普朗克常数、精细结构常数
- 散射参数：α粒子电荷数、金核电荷数、库仑常数
- 实验参数：α粒子动能（7.7 MeV）

#### 经典散射.py
经典散射核心计算模块（数值模拟方法）：
- `single_classical_scatter()`: 单个粒子的经典散射数值模拟
  - 使用欧拉方法积分相对论运动方程
  - 计算库仑力作用下的粒子轨迹
  - 返回散射角
- `batch_classical_scatter()`: 批量处理多个粒子的经典散射

#### 量子散射.py
量子散射核心计算模块（波包演化方法）：
- `init_wavepacket()`: 初始化高斯波包
- `V_coulomb()`: 库仑势场计算
- `evolve_wavepacket()`: 分裂步傅里叶方法演化波包
- `evolve_and_get_snapshots()`: 演化并保存多个时间帧
- `single_quantum_scatter()`: 单个波包的量子散射模拟
- `batch_quantum_scatter()`: 批量处理多个粒子的量子散射

#### 抽样.py
蒙特卡洛粒子抽样模块：
- `monte_carlo_sample()`: 随机抽样粒子参数（能量、横向位置、横向动量）

### 3. 分析与可视化模块

#### 绘图.py
数据可视化模块：
- `plot_single_wavepacket_evolution()`: 绘制单个波包演化图
- `plot_wavepacket_evolution()`: 绘制多个波包演化对比图
- `plot_theta_b_relation()`: 绘制散射角与碰撞参数关系图
- `plot_summary_table()`: 生成数据汇总表（Markdown格式）

#### 波包热力图.py
波包演化动画生成模块：
- 从量子散射数据中提取波包演化过程
- 生成热力图展示波包密度随时间的变化

## 运行说明

### 运行主程序

```bash
cd 经典散射和量子散射
python main.py
```

主程序执行流程：
1. 验证网格Nyquist条件
2. 抽样30个粒子（碰撞参数范围：10~125 fm）
3. 经典散射计算（数值积分运动方程）
4. 量子散射计算（波包演化，分裂步傅里叶方法）
5. 生成图二：散射角与碰撞参数关系图
6. 生成数据汇总表

### 生成波包演化图

```bash
cd 经典散射和量子散射
python 波包热力图.py
```

生成5个不同碰撞参数的波包演化图，展示量子散射过程。
